# How To Run

* Clone the repo
* To start Frontend
    * Go to the frontend directory (cd frontend)
    * npm install
    * npm run dev
* To start Backend
    * Go to the backend directory (cd backend)
    * npm install
    * go to .env_sample do as it is said there then rename it to .env
    * node index.js
